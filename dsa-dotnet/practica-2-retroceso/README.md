# ed-retroceso-cs
Práctica de solución de problemas en forma recursiva mediante la técnica de retroceso (backtracking).

# Ejecución

Para ejecutar el código, entra al directorio ```Laberinto``` y ejecuta ahí:
```
dotnet run
```

# Creación del proyecto

Se incluyen aquí las instrucciones que fueron utilizadas para la creación de este proyecto.  Ya no es necesario ejecutarlas.

Desde el directorio raíz:
```
dotnet new avalonia.app -o Laberinto -f net7.0
```
