**Observaciones**

* Para sólo compilar uno de los projects (ej: ComplejidadMain), entra en su directorio y ejecuta:
```
dotnet build
```

* Para ejecutar las pruebas unitarias usa desde el directorio _ed\_complejidad\_cs_.
```
dotnet test ComplejidadNPruebas/ComplejidadNPruebas.csproj
```

* Para ejecutar implementa el archivo _Main.cs_ con un método _Main_ dentro de _ComplejidadMain_ en el espacio de nombres _Análisis_ y utiliza
```
dotnet run --project ComplejidadMain/ComplejidadMain.csproj
```
