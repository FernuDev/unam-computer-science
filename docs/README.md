## Cómo Usar Esta Documentación

Esta documentación está diseñada para:

1. **Estudio y Repaso**: Conceptos teóricos explicados de manera clara
2. **Referencia Rápida**: Complejidades, casos de uso y comparaciones
3. **Complemento a las Prácticas**: Contexto teórico para las implementaciones

Cada documento incluye:
- Definición formal del concepto
- Propiedades y características
- Complejidades temporales y espaciales
- Casos de uso y aplicaciones
- Ventajas y desventajas
- Relación con otros conceptos
