## Sobre este Repositorio

Este repositorio contiene todas las implementaciones, ejercicios y proyectos desarrollados durante las materias de **Estructuras de Datos y Algoritmos**, **Simulación y Modelado de Software** y **Física Computacional** en la Facultad de Ciencias de la UNAM.

El objetivo es mantener un registro organizado del aprendizaje, compartir soluciones y servir como referencia para futuros estudiantes.

## Instalación y Uso

### Requisitos Previos

**Para Estructuras de Datos (.NET/C#):**
```bash
# Instalar .NET SDK (versión 6.0 o superior)
# Descarga desde: https://dotnet.microsoft.com/download
```

**Para Simulación y Modelado (C++):**
```bash
# C++11 o superior y CMake
# macOS: 
brew install cmake

# Linux:
sudo apt-get install cmake g++ libgtest-dev

# Windows: 
# Instalar Visual Studio con C++ tools
# Instalar CMake desde: https://cmake.org/download/
```

**Para Física Computacional:**
```bash
# Python 3.x
pip install -r fisica-computacional/requirements.txt

# Compilador de Fortran (gfortran)
# macOS: brew install gcc
# Linux: sudo apt-get install gfortran
```

### Compilar y Ejecutar

**Proyectos C#:**
```bash
cd dsa-dotnet/practica-X-nombre
dotnet build
dotnet run
```

**Proyectos C++ (CMake):**
```bash
cd software-modeling/ejercicio_X
mkdir build && cd build
cmake ..
make

# Ejecutar programa principal
./main

# Ejecutar pruebas unitarias
make test
# o directamente
./test/UnitTestingSolution
```

**Programas Fortran:**
```bash
cd fisica-computacional/tarea_X
gfortran -o programa archivo.f90
./programa
```

**Notebooks Python:**
```bash
cd fisica-computacional/tarea_X
jupyter notebook
```
